Given an OTU table and a taxonomy file, micca-taxonomy builds an OTU table with the taxomomy information and several OTU tables for each taxonomic level. OTU counts are summed together if they have the same consensus at the considered level.

$ micca-levels --help
usage: micca-levels [-h] [-v] [-o DIR] otutable taxonomytable

Given an OTU table and a taxonomy file, micca-taxonomy builds an OTU table with
the taxomomy information and several OTU tables for each taxonomic level. OTU 
counts are summed together if they have the same consensus at the considered
level.

positional arguments:
  otutable              OTU table file
  taxonomytable         taxomomy file

optional arguments:
  -h, --help            show this help message and exit
  -v, --version         show program's version number and exit
  -o DIR, --output-dir DIR
                        output directory (default .)

Example:

 $ micca-taxonomy otu_table.txt taxonomy.txt -o otu_table_taxonomy
