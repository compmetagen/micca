Plots OTUs relative abundaces for each sample from an OTU table.

$ micca-plot-abundance --help
usage: micca-plot-abundance [-h] [-v] [-s SAMPLE_DATA] [-t TARGET]
                            otutable output

micca-plot-abundance plots OTUs relative abundaces for each sample from an
OTU table.

positional arguments:
  otutable              input OTU table file
  output                output image file (.png, .pdf, .ps, .eps, .svg). The
                        image type is autodetected from the file extension

optional arguments:
  -h, --help            show this help message and exit
  -v, --version         show program's version number and exit
  -s SAMPLE_DATA, --sample-data SAMPLE_DATA
                        sample file
  -t TARGET, --target TARGET
                        sample-data column to be used as label

Example:

 $ micca-plot-abundance otu_table.txt image.png
