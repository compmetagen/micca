Rarefies an OTU table with a depth of perc % of the least abundant sample.

$ micca-rarefy --help
usage: micca-rarefy [-h] [-v] [-p PERC] [-s SEED] input output

micca-rarefy rarefies an OTU table with a depth of perc % of the least
abundant sample.

positional arguments:
  input                 input OTU table file
  output                output OTU table file

 $ micca-rarefy otu_table.txt rarefied.txt -p 95
